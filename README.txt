Program for Generating EMS and Registered Mail Numbers

This program is designed to assist in generating EMS and Registered Mail numbers according to the postal format. Users can input data to create a range of numbers and use them as needed.

The user inputs are as follows:

Two starting letters: such as RK, RR, TH, or other letters used in EMS and Registered Mail systems.
Starting number: The 8-digit EMS or Registered Mail number that serves as the starting point for generating the list.
Ending number: The 8-digit EMS or Registered Mail number that serves as the ending point for generating the list.
Two ending letters: such as TH, US, or other letters used in EMS and Registered Mail systems.
Once all information is entered, the program will display the result and allow it to be saved in both Excel and TXT formats, making it easy for further use.

This program can be used for both EMS and Registered Mail systems according to postal standards.

This program is Open Source, and you are free to download, modify, and distribute it under open-source licensing terms.

Programme pour générer des numéros EMS et Courrier Recommandé

Ce programme est conçu pour aider à générer des numéros EMS et des numéros de courrier recommandé conformément au format postal. Les utilisateurs peuvent entrer des données pour créer une plage de numéros et les utiliser selon leurs besoins.

Les informations à entrer par l'utilisateur sont les suivantes :

Deux lettres de départ : telles que RK, RR, TH ou d'autres lettres utilisées dans les systèmes EMS et Courrier Recommandé.
Numéro de départ : Le numéro EMS ou Courrier Recommandé à 8 chiffres servant de point de départ pour générer la liste.
Numéro de fin : Le numéro EMS ou Courrier Recommandé à 8 chiffres servant de point de fin pour générer la liste.
Deux lettres de fin : telles que TH, US ou d'autres lettres utilisées dans les systèmes EMS et Courrier Recommandé.
Une fois toutes les informations saisies, le programme affichera le résultat et permettra de le sauvegarder aux formats Excel et TXT, facilitant ainsi son utilisation ultérieure.

Ce programme peut être utilisé pour les systèmes EMS et Courrier Recommandé selon les normes postales.

Ce programme est Open Source, et vous êtes libre de le télécharger, modifier et distribuer sous les termes de la licence open-source.
